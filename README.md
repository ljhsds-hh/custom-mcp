# custom-mcp

企业级自定义 MCP (Model Context Protocol) 服务集合。

## 项目概述

本仓库致力于构建高质量、可复用的 MCP 服务，为 AI 编程助手（Claude Code、Cursor、Trae 等）提供标准化的外部工具集成。每个 MCP 服务独立封装、独立发布，确保模块化与可维护性。

所有服务遵循统一的设计原则：

- **安全性** — API Key 等敏感信息通过环境变量注入，不硬编码，不泄露
- **可靠性** — 异步连接池、自动重试、完善的错误处理与分类
- **规范性** — PyPI 标准发布格式，`uvx` 一行配置即用
- **可扩展性** — 模块化架构，便于后续维护与功能迭代

## 服务列表

## 快速使用

每个服务均支持 `uvx` 一行配置，无需手动克隆或安装依赖。详见各服务的 README。
